# matthewmincher.dev

Personal website built with [Astro](https://astro.build), React, and Tailwind CSS 4.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Builds the static site and generates CV PDFs via Puppeteer.

## Deployment

Deployed to Cloudflare Pages via GitHub Actions on push to `main`.
